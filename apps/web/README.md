# AI Code Review Platform Web

Next.js web application for the AI Code Review Platform. It handles the public site, authentication callbacks, onboarding, repository integrations, dashboards, analysis views, team workflows, and settings.

## Development

Run from the repository root:

```sh
pnpm dev --filter web
```

Build the web app:

```sh
pnpm build --filter web
```

Run type checks:

```sh
pnpm check-types --filter web
```
